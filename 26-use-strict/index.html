Understanding "use strict" in JavaScript

This interactive page helps you understand the effects of JavaScript's strict mode by running different test cases.

Global Strict Mode

"use strict"; try { undeclaredVar = 10; } catch (e) { console.log(e.message); }

Function-Level Strict Mode

function strictFunction() { "use strict"; try { y = 20; } catch (e) { console.log(e.message); } } strictFunction();

Duplicate Object Property

const obj = { name: "Alice", name: "Bob" };

Duplicate Function Parameters

function dup(a, a) { "use strict"; return a; }

`this` in Function

function checkThis() { "use strict"; console.log(this); } checkThis();

Eval in Strict Mode

eval("var z = 30;"); console.log(typeof z);

Read-Only Property

const obj = {}; Object.defineProperty(obj, "name", { value: "Alice", writable: false }); obj.name = "Bob";

Check if Strict Mode is Enabled

function isStrict() { return (function() { return !this; })(); } console.log(isStrict());
