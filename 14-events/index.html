JavaScript Event Handling Examples

1️⃣ Inline Event Handling (onclick)

This is the simplest way to handle events. The event is written directly inside the HTML element using the onclick attribute.

2️⃣ DOM Event Handling

Here, JavaScript assigns an event handler directly to a DOM element using document.getElementById().onclick.

3️⃣ addEventListener

The recommended method to handle events. It allows multiple event listeners on the same element.

4️⃣ Input Events

The input event fires every time the user types something, while the change event fires when the input loses focus.

5️⃣ Mouse Events

Events triggered when the user interacts with the mouse, such as mouseenter and mouseleave.

6️⃣ Keyboard Events

The keydown event fires when a key is pressed, and keyup fires when the key is released.

7️⃣ classList.add()

Used to add or remove CSS classes dynamically using JavaScript.

8️⃣ document.getElementById().style

Allows direct manipulation of an element’s style properties.
